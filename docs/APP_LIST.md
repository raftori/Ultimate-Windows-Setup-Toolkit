## Adding Custom Applications

To add your own applications, edit `configs/apps.json`:

```json
{
    "Name": "Your App Name",
    "WingetId": "Publisher.AppName",
    "ChocoId": "package-name",
    "Description": "Brief description",
    "Essential": false
}
```

### Finding Package IDs

**Winget:**
```powershell
winget search "app name"
```

**Chocolatey:**
- Search at https://community.chocolatey.org/packages
- Or use: `choco search "app name"`

### Verifying Package IDs

Before adding, verify the ID works:
```powershell
# Winget
winget show Publisher.AppName

# Chocolatey
choco info package-name
```
